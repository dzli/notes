# Code Indexing

Systems and tools that parse, index, and make searchable the symbols, structure, and relationships within source code repositories. Distinct from full-text search over code; code indexing involves AST parsing, symbol extraction, dependency graph construction, and semantic queries such as call hierarchies, blast-radius analysis, and dead-code detection.
